# Evolutionary-feature-selection
This is the public notice for the source code of papers published by Yu-Yang Zhang, a PhD student at Southeast University. This is a repository containing the source code corresponding to the author's published literature in the field of evolutionary feature selection.

Paper 1: Zhang Y Y, Jia B B, Zhang M L. Evolutionary Dimension-Specific Feature Selection for Multi-Dimensional Classification[J]. IEEE Transactions on Pattern Analysis and Machine Intelligence. 2025. Code at: <https://palm.seu.edu.cn/zhangml/>

Paper 2: Zhang Y Y, Jia B B, Zhang M L. Evolutionary Classifier Chain for Multi-Dimensional Classification[C]//Proceedings of the AAAI Conference on Artificial Intelligence. 2025, 39(21): 22641-22649. Code at: <https://palm.seu.edu.cn/zhangml/>

Paper 3: Liang J, Zhang Y, Qu B, et al. A multiform optimization framework for multiobjective feature selection in classification[J]. IEEE Transactions on Evolutionary Computation, 2023, 28(4): 1024-1038. Code at: <https://www5.zzu.edu.cn/cilab/Code.htm>

Paper 4: Liang J, Zhang Y, Chen K, et al. An evolutionary multiobjective method based on dominance and decomposition for feature selection in classification[J]. Science China Information Sciences, 2024, 67(2): 120101. Code at: <https://www5.zzu.edu.cn/cilab/Code.htm>

Paper 5: Liang J, Zhang Y Y, Qu B, et al. A Multiform Framework for Multiobjective Feature Selection in Unbalanced Classification: Combining Oversampling and Cost-Sensitive Learning[J]. IEEE Transactions on Systems, Man, and Cybernetics: Systems, 2025. Code at: <https://www5.zzu.edu.cn/cilab/Code.htm>

Paper 6: Liang J, Yang J, Yue C, et al. A Joint-Encoding Evolutionary Algorithm for Multimodal Multiobjective Feature Selection in Classification[J]. IEEE Transactions on Evolutionary Computation, 2025. Code at: <https://www5.zzu.edu.cn/cilab/Code.htm>
